# Paradox-Localizer
Paradox Localizer 1.2 is a powerful tool for automating the translation of Paradox game localization files (HOI4, EU4, Stellaris). Auto-Verify mode streamlines the translation of thousands of strings via Google Translate. It's the perfect solution for mod developers, turning weeks of monotonous work into hours of automated work.


# USAGE
python localiser.py -path "path_to_file" -ref "path_to_reference_file(optional)" -start 0 <br>
-path is necessary <br>
-ref is optional (use it only if you create a translationn based on other translation mod) <br>
-start is optional (it means the line number from which the program will open) <br>
