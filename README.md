# Paradox-Localizer
Paradox Localizer is a powerful tool for automating the translation of Paradox game localization files (HOI4, EU4, Stellaris). Auto-Verify mode streamlines the translation of thousands of strings via Google Translate. It's the perfect solution for mod developers, turning weeks of monotonous work into hours of automated work.

# USAGE (.exe)
1 variant <br>
Just open .exe and move .yml to window <br>
2 variant <br>
Drug&drop your .yml on .exe <br>

# USAGE (OLD, ONLY IN ORIGINAL .py FILE)
python localiser.py "URL" "path_to_file" -ref "path_to_reference_file(optional)" -start 0 <br>
-ref is optional (use it only if you create a translationn based on other translation mod) <br>
-start is optional (it means the line number from which the program will open) <br>
